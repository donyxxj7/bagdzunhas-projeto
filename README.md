# 💅 Ana Bagdzinski - Site Profissional de Manicure

![Prévia do Site](https://i.imgur.com/S0IYqVx.jpeg) Este é o repositório do site profissional desenvolvido para a manicure e nail designer Ana Bagdzinski. O projeto é uma Single-Page Application (SPA) moderna, totalmente responsiva e construída com as tecnologias mais recentes do ecossistema React.

**[➡️ Visite a versão ao vivo!](https://bagdzunhas.store)** ---

## ✨ Funcionalidades Principais

O site foi projetado para ser não apenas um portfólio, mas uma ferramenta completa de negócios, com foco na experiência do usuário e na conversão de clientes.

* 🎨 **Design Moderno e Responsivo:** Interface elegante com gradientes sofisticados, animações suaves e layout adaptável para desktops, tablets e celulares.
* 🌗 **Modo Claro e Escuro (Dark Mode):** Um tema escuro deslumbrante que pode ser alternado com um clique, proporcionando conforto visual.
* 📸 **Galeria de Fotos Interativa:** Portfólio com um modal de visualização em carrossel para exibir os trabalhos em alta resolução.
* 🔄 **Seção "Antes e Depois":** Um carrossel dedicado para mostrar o impacto e a qualidade das transformações, uma poderosa ferramenta de prova social.
* 🌟 **Seção de Depoimentos:** Exibe feedbacks de clientes satisfeitas em um carrossel, aumentando a confiança e credibilidade.
* ❓ **FAQ Interativo:** Uma seção de Perguntas Frequentes com um componente "acordeão" para responder às dúvidas mais comuns de forma organizada.
* 📱 **Integração com WhatsApp:**
    * Formulário de **Agendamento** que gera uma mensagem pré-formatada.
    * Formulário de **Indicação** para o programa de fidelidade.
* 🚀 **Otimizado para SEO:** Meta tags (`title` e `description`) cuidadosamente elaboradas para melhorar o ranqueamento em buscadores como o Google.

## 🚀 Começando

Siga os passos abaixo para rodar o projeto localmente em sua máquina.

### Pré-requisitos

Você precisará ter o [Node.js](https://nodejs.org/) (versão 18 ou superior) e o [pnpm](https://pnpm.io/) instalados.

Para instalar o pnpm (caso não tenha), rode no seu terminal:
```bash
npm install -g pnpm
```

### Instalação

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/seu-usuario/nome-do-repositorio.git](https://github.com/seu-usuario/nome-do-repositorio.git)
    cd nome-do-repositorio
    ```

2.  **Instale as dependências:**
    ```bash
    pnpm install
    ```

3.  **Configure as Variáveis de Ambiente:**
    * Crie um arquivo chamado `.env` na raiz do projeto.
    * Adicione a seguinte linha, substituindo pelo número de WhatsApp desejado:
        ```
        VITE_WHATSAPP_NUMBER=5547984887462
        ```

### Rodando o Projeto

Para iniciar o servidor de desenvolvimento, execute:
```bash
pnpm run dev
```
Abra [http://localhost:5173](http://localhost:5173) no seu navegador para ver o resultado.

## ⚙️ Configuração

Além do número de WhatsApp no `.env`, outras informações podem ser facilmente alteradas diretamente no código:

* **Link do Instagram:** No arquivo `src/App.jsx`, localize o `<footer>` e altere o link no botão do Instagram.
* **Textos e Imagens:** Todas as informações de serviços, depoimentos, FAQ e imagens do portfólio estão centralizadas como arrays de objetos no início do arquivo `src/App.jsx`, facilitando a edição e adição de novos conteúdos.
