# Portfólio Vitrine para Manicure

![Status do Projeto: Concluído](https://img.shields.io/badge/status-conclu%C3%ADdo-brightgreen)

Um site de portfólio de página única, elegante e funcional, desenvolvido para a profissional manicure Ana Bagdzinski. O projeto serve como um cartão de visitas digital, projetado para exibir trabalhos, detalhar serviços e, principalmente, capturar clientes através de funcionalidades de agendamento e indicação.

## ✨ Funcionalidades Principais

Este projeto foi construído com foco na experiência do usuário e na conversão de visitantes em clientes.

* **💅 Galeria de Trabalhos:** Uma grade visualmente atraente e responsiva para exibir as melhores fotos dos trabalhos da profissional.
* **📝 Seção de Serviços:** Cards detalhados que explicam cada serviço oferecido, como Manicure, Pedicure e Spa dos Pés.
* **💲 Tabela de Preços:** Uma tabela clara e que se adapta a dispositivos móveis, facilitando a consulta de valores pelos clientes.
* **🗓️ Formulário de Agendamento:** Permite que os clientes preencham seu nome, contato, serviço desejado e data/hora preferencial. Ao enviar, o formulário gera uma mensagem de WhatsApp pré-formatada, agilizando o processo de agendamento.
* **🏆 Formulário de Indicação "Indique e Ganhe":** Uma ferramenta de marketing onde clientes existentes podem indicar amigas. O formulário também gera uma mensagem de WhatsApp, facilitando o controle da promoção pela profissional.
* **📱 Design Totalmente Responsivo:** O layout se adapta perfeitamente a desktops, tablets e celulares, garantindo uma ótima experiência em qualquer dispositivo.
* **💬 Botão Flutuante do WhatsApp:** Para contato direto e rápido.

## 💻 Tecnologias Utilizadas

O projeto foi construído utilizando as tecnologias fundamentais da web, com foco em código limpo e performance.

* **HTML5:** Para uma estrutura semântica, organizada e acessível.
* **CSS3:** Utilizando recursos modernos como **Variáveis CSS** para fácil manutenção de temas, **Flexbox** e **Grid Layout** para a construção de layouts complexos e responsivos.
* **JavaScript (ES6+):** Para toda a interatividade da página, incluindo a validação de formulários e a geração dinâmica de links para o WhatsApp.

## 🚀 Como Executar o Projeto Localmente

Como este é um projeto front-end estático, você não precisa de nenhuma ferramenta complexa para executá-lo.

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/donyxxj7/bagdzunhas-projeto.git
    ```

2.  **Navegue até a pasta do projeto:**
    ```bash
    cd bagdzunhas-projeto
    ```

3.  **Abra o arquivo `index.html`:**
    Basta abrir o arquivo `index.html` no seu navegador de preferência.

    > **💡 Dica de Profissional:** Para uma melhor experiência de desenvolvimento, recomendo usar a extensão **Live Server** no Visual Studio Code. Ela cria um servidor local e atualiza a página automaticamente sempre que você salva uma alteração no código.
